# Caption Style Reference

產生 Facebook Page 文案時的選用風格參考。SKILL.md 對文案的指引保持中性；只有在使用者選擇採用某種特定風格（或個人偏好指向這裡）時，才參考本檔。

## Travel / nature / cinematic（繁體中文，溫柔且詩意）

指引：

- 用繁體中文。
- 步調放慢、語氣溫柔、帶點詩意——不要過度誇張。
- 最多 2～4 個短段落。
- 視情況加 3～6 個相關 hashtag。
- 避免那種一聽就是 AI 寫的制式推銷話術，例如「開啟一場視覺盛宴」。

範例：

```text
午後的水邊，時間好像慢了下來。

把這段安靜的小旅行剪成一支可愛又療癒的短片，記錄那些不特別喧鬧、卻讓人想多停留一下的片刻。
希望你也能在這 1 分鐘裡，感覺到一點點放鬆與溫柔。

#旅行日記 #療癒風景 #慢步調生活
```

## Adding new styles

當使用者想要另一種會重複使用的風格（例如有力的推銷、美食評論、產品上市），在這裡新增一個段落，包含：

- 語氣描述
- 長度／格式限制
- hashtag 慣例
- 一個具體範例

個別使用者的個人偏好請放在他們自己的 CLAUDE.md／memory，而不是放這裡；本檔是給可共用的參考風格用的。
